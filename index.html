    📖 원카드 규칙

    마지막 버려진 카드와 무늬 또는 숫자(종류)가 같은 카드만 낼 수 있어요.

    공격 카드

    A = +3장 / 2 = +2장 / 흑백 조커 = +5장 / 컬러 조커 = +7장 (조커 공격은 방어 불가)

    공격 이어내기 (같은 숫자 OR 같은 무늬)

    예) 하트2 → 하트A → 클로바A → 클로바2

    흑백 조커: 직전이 흑백조커·클로바·스페이드일 때만 가능

    컬러 조커: 직전이 흑백조커·다이아·하트일 때만 가능

    컬러 조커 다음엔 흑백 조커만 이어낼 수 있어요.

    3 방어

    마지막 공격 카드와 같은 무늬의 3을 내면 공격을 전부 무효화! (조커 공격은 불가)

    특수 카드

    J = 다음 사람 건너뜀 · Q = 방향 반전 · K = 한 번 더 · 7 = 무늬 변경

    원카드

    카드 1장 남을 때 반드시 "원카드!" 선언. 깜빡하면 상대가 지적해서 +1장 벌칙!

    파산

    손패가 24장 이상 되면 파산(패배). 마지막 생존자가 승리.
